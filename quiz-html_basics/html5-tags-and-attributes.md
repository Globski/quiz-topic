# Question: What is the latest version of HTML, and what are its two main components?
**Answer:**  
The latest version of HTML is **HTML5**. The two main components of HTML are **Tags** and **Attributes**.

- **Tags** define the elements on the webpage. For example, `<a>` is a tag used to create a hyperlink.
- **Attributes** provide extra information about tags. For example, the `href` attribute specifies the link destination in an `<a>` tag.

Example:  
```html
<a href="https://www.example.com">Visit Example</a>
